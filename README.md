# SF Express Tracking Dashboard

Local dashboard for SF Express order and tracking relationships.

## Guides

- Korean usage and feature guide: [docs/SF_TRACKING_DASHBOARD_GUIDE_ko.md](docs/SF_TRACKING_DASHBOARD_GUIDE_ko.md)
- Windows packaging and update guide: [docs/WINDOWS_PACKAGING_AND_UPDATE_ko.md](docs/WINDOWS_PACKAGING_AND_UPDATE_ko.md)

## Commands

- `make install`
- `make run-backend`
- `make run-frontend`
- `make build`
- `make test`

## Structure

- `backend/`: FastAPI, SQLAlchemy, APScheduler, Alembic
- `frontend/`: React, Ant Design, Recharts
- `packaging/`: PyInstaller and Inno Setup assets
